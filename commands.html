Commands

A full list of every command the bot has.



1. /help - Displays a help command.
2. /cmds - Shows a full list of all valid commands.
3. /kick (user) (reason) - Kicks a user from the server.
4. /ban (user) (reason) - Bans a user from the server.
5. /setup - Sets up the server for Ares.
6. /unsetup - Removes all channels created by the bot.
7. /clear (ammount) - Clears a channel with a inputed number.
8. /mute (user) (reason) (time) - Mutes a server member.
9. /unmute (user) - Unmutes a server member.
10. /unban (user ID) - Unbans a user from the server.
11. /addrole (user) (role) - Adds a role to a user.
12. /removerole (user) (role) - Removes a role to a user.
13. /nickname (user) (name>) - Changes a users nickname.
14. /ping - Displays the bots ping.
15. /message (user) (text) - Sends a personal DM to a user.
16. /embed (channel) (options) - Sends a custom embed in a desired channel.
17. /dmrole (role) (message) - Sends a personal message to everyone with the role.
